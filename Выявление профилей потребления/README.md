**Название**: Выявление профилей потребления\
**Описание проекта**: На основе полученных данных от магазина «Пока все ещё тут» сегментировать пользователей и разработать персональные предложения для клиентов\
**Выводы**: Были иследованы все покупки, товары были разделены на 6 групп и после этого пользователи сегментированы на 4 группы. Для каждой группы были найдены популярные группы и посоветованы персональные предложения для каждого кластера
